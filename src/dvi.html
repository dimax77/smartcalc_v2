SmartCalc_v2.0

О программе

SmartCalc_v2.0 позволяет выполнять вычисления базовых арифметический функций, тригонометрический функций, строить графики, а также допоплнительно предоставляет возможность рассчитать кредит и депозит.

Программа позволяет

    • Вычислять основные операции - +, -, ;, /, возведение в степень.
    • Вычислять cos, sin, tg, arccos, arcsin, arctg.
    • Вычислять ln, log.
    • Строить графики функций.
    • Рассчитывать платежи по кредиту.
    • Рассчитывать доходность вклада.

Установка программы

Для того, чтобы начать использовать SmartCalc_v2.0 вам нужно использовать команду make install.

Открыть программу

Для того, чтобы открыть SmartCalc_v2.0 вам нужно использовать команду make open.

Удаление программы

Для удаления программы вы можете использовать команду make uninstall.
